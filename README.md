# Portal Estudiantil San Ignacio de Loyola

Este es el frontend para el portal estudiantil, desarrollado con tecnologías web modernas.

## Stack de Tecnologías

*   **Framework:** [React](https://react.dev/)
*   **Bundler:** [Vite](https://vitejs.dev/)
*   **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
*   **Estilos CSS:** [Tailwind CSS](https://tailwindcss.com/)
*   **Enrutamiento:** [React Router](https://reactrouter.com/)

## Primeros Pasos

Sigue estas instrucciones para levantar el proyecto en tu máquina local para desarrollo y pruebas.

### Prerrequisitos

Asegúrate de tener instalado [Node.js](https://nodejs.org/) (se recomienda la versión LTS).

### Instalación

1.  Clona el repositorio (si aún no lo has hecho).
2.  Navega al directorio del frontend:
    ```sh
    cd frontend/portal-estudiantil-san-ignacio-loyola
    ```
3.  Instala las dependencias del proyecto:
    ```sh
    npm install
    ```

## Scripts Disponibles

En el directorio del proyecto, puedes ejecutar los siguientes comandos:

### `npm run dev`

Inicia la aplicación en modo de desarrollo. Abre un servidor local (generalmente en `http://localhost:5173`) para verla en tu navegador.

### `npm run build`

Compila la aplicación para producción en la carpeta `dist`. Revisa que no haya errores de TypeScript y empaqueta el código de forma optimizada.

### `npm run lint`

Ejecuta el linter (ESLint) para analizar el código en busca de problemas y forzar un estilo consistente.

### `npm run preview`

Levanta un servidor local para previsualizar la versión de producción que se encuentra en la carpeta `dist`.
